# TP Reflection Dynamic Loading

Ce projet Maven illustre l'utilisation de la réflexion en Java pour charger dynamiquement des classes, injecter des dépendances et invoquer des méthodes. Il s'agit d'un exercice pédagogique pour comprendre la programmation dynamique et l'injection de dépendances sans framework externe.
